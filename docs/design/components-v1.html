10

Token reference

CSS variables · Tailwind mapping

All tokens are CSS custom properties on :root (global) with theme overrides on .theme-light / .theme-dark for the spec, and on @media (prefers-color-scheme: dark) for the live pages.

TokenLight valueDark valueTailwind keyUsage
Color — Background
--bg oklch(97% 0.018 70) oklch(16% 0.012 80) bg-bg Page background, summary panel
--surface oklch(94% 0.016 70) oklch(13% 0.010 80) bg-surface Cards, sheets, sidebar
Color — Foreground
--fg oklch(25% 0.022 70) oklch(91% 0.014 75) text-fg Primary text, headings
--muted oklch(44% 0.020 65) oklch(66% 0.016 70) text-muted Secondary text, captions, meta
Color — Accent
--accent oklch(58% 0.13 35) oklch(62% 0.14 35) text-accent bg-accent Primary CTA, pull-quote rule, active states. Used ≤ 2× per screen.
Color — Border
--border oklch(87% 0.014 70) oklch(24% 0.012 70) border-border All hairline borders, dividers
Typography
--display 'Newsreader', Georgia, serif font-display Headlines, wordmark, pull quotes, summaries. Min 22px on mobile.
--body 'Inter', -apple-system, system-ui, sans-serif font-body UI labels, body copy, buttons
--mono 'JetBrains Mono', ui-monospace, Menlo, monospace font-mono Eyebrows, category codes, timestamps, admin IDs
Radii
--r-card 12px rounded-card Feed headline tile, story cards
--r-sheet 16px rounded-sheet Bottom sheet top corners
--r-pill 999px rounded-full All pill-shaped buttons, category pills, badges
Shadow
--shadow-card 0 10px 28px -16px (warm/dark at 30–55%), 0 1px border shadow-card Headline overlay tile, feed card
--shadow-sheet 0 -16px 40px -20px (upward, bottom sheet) shadow-sheet Bottom sheet lift
--shadow-drop 0 4px 20px -8px + 1px ring shadow-drop Dropdowns, install banner, popovers
Accent budget rule
One accent, used at most twice per screen. Acceptable uses: primary CTA button fill, pull-quote left-rule, active category pill background, eyebrow category dot, Publish action in admin. Never gradient the accent. Never use it as a background on large surfaces.
